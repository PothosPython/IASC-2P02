IASC 2P02 - Liv Rowbottom - Week 4

Week 4 - Flatlands

Specifications

  • The scene must contain a plane mesh. The plane must appear as a wireframe when the page first loads.
  • The scene must contain a mesh that intersects the plane (for example, a sphere, cube, torus, etc.).
  • The scene must contain working OrbitControls.
  • The scene must contain a working UI using lil-gui. This UI must let the user change the position of the mesh that intersects the plane.

Challenges: 1. add a rotation toggle (on/off + speed slider) 2. come up w/ your own flatland scene to tell a story
